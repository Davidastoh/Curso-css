El objetivo de crear mi archivo de estilos de css es para separara la estructura del diseño

  • Asi tendremos mi archivo.html solo solo para la estructura elementos ,etiquetas y contenido
  • Y mi archivo de css solo opara los selectores y propiedaes de diseño que le asignare a cada etiqueta

Para relacionar un archivo html con su archivo css ( Hojas de estilo).Debemos hacer uso de nuestra etiqueta link en la cabezera del documento, y un atributo href le indicamos la ruta donde se encuentro el archivo
